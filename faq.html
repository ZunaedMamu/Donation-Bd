Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface that represents a web page's structure as a tree of objects, allowing manipulation of HTML and XML documents.

How do you select an element from the DOM?

You can select an element from the DOM using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), or querySelector().

What is event Delegation in the context of DOM and why is it useful?

How do you manipulate an element's attribute and style using DOM?

You manipulate an element's attributes using setAttribute() and getAttribute(), and you change its style using the style property (e.g., element.style.color = 'red';).
